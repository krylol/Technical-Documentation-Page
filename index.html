CSS selectors
  • Class selector:
    Using this selector allow you to pick an element that has a class. You use . and then the class name to select it
    class="intro" .intro{font-size:25px;}

  • ID selector:
    Using this selector allow you to pick an element that has an ID. You use # followed by the ID name.
    id=intro #intro{color:blue}

  • Element selector: With this selector you can pick all the elements of the same kind, you can edit all the divs the p or even the a link.
    p{background-color:black;}

  • Selectors priority:
    1. Inline styles An inline style is attached directly to the element to be styled.
      h1 style="color: #ffffff;"

    2. ID element the ID is a unique identifier for the page elements. This element should be used only one time you shouldn't use the same ID for different elements.
      #intro #play #changing

    3. Class elements, you can assign the same class to your div or p element.
      div class="color" p class="color" .color
    4. Elements and pseudo-elements - This category includes element names and pseudo-elements.
      h1, p, div,

    5. You can use the !important to make the style you want have the priority even if it shouldn't have it.
      p id=thing p{color: "red"!important;} #thing{color="green"}
      If you try this the element will be red even when the ID has priority using important overwrites it and makes the p element more important.

Ways to change font

There are a few ways to change font type, size make it bold or italic. I will only talk about the most used, the way to change font, the way to change the size of the words and you will learn to change a word type to bold or make it only a bit bold if need it.

We will start with changing the size of the letters. First we need to choose the element we want to change and then use this format.
p class="thing" .thing{font-size:20px;}
With this you can change the font to 20px. You can input any size you want.

The next way is to change the font. With this you can use the font you like or the font you think will suit your page better.
p id="thing" #thing{font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif}
The first font you write is the one you want to use first. If the browser you are using don't have that font available it will take the second you wrote. In this case it will try to use Franklin but if Franklin is unavailable it will use arial narrow, if that one it's not in the browser it will take te next one. Usually browsers have all fonts but it's a nice a way to prevent your page from not looking the way you want.

The last style we are going to learn is the way to make letters or words bolder. You can use it on a few words, all the text or only the word you want to.
p id=thing #thing{font-size: bold}
With this the letter will be bold. You can try it yourself and then change bold to bolder or to a number from 100 to 900 to see how it changes.

Playing with color

One of the easiest ways to make your page look beautiful. After using color you need to adjust margins, paddings and fonts but this is the basic way to make it stylier.
Here we got a few choices but we will learn the more common ones.

  • First is the way to change the background color.
    p class=thing .thing{background-color: antiquewhite;}
    With this all the background will be antiquewhite. You can use any color you want and learn more about colors in the mozilla documentation page.
  • You can also change the font colors. It's easy and will make some parts of the web beautiful.
    p p{color:red}
    An easy way to make all the letters of a p element in red color.

Creating boxes and borders

Creating borders can isolate elemnts or remark them. Is very useful for creating bars, links, or any other thing you want to remark.
p id=thing #thing{border-style: solid;}
With the color styling we learned before we can make those borders look any way we want. With this we created and isolated element, that is easy to view.

How to center the elements

This is something you see in every page. All the elements are in the middle, in their own background color, centered and styled. It's something basic because only a few pages like this one are not centered and take all the space they can. Learning this will make a page styler with 6 lines of codes.


div class=container .thing{max-width: 500px; maring: auto;background-color:silver;}

With this few lines you can put all the content in the middle of the page. You should use a div with and id of container, this div should be around the elements you want to center, and then you just apply the CSS style over.
